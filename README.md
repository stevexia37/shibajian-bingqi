# 十八般兵器 · 粒子演绎

> 明·武备志版本 | Three.js + GPU 粒子系统

黑色空间中，6 万粒子按明代《武备志》十八般兵器（刀、枪、剑、戟、戈、矛、盾、斧、钺、鞭、锏、锤、叉、镗、棍、槊、棒、流星锤）依次汇聚、成型、击碎、散开。

## 访问

开启 GitHub Pages 后访问：

    https://stevexia37.github.io/shibajian-bingqi/

## 启用 Pages（首次）

1. 进入 https://github.com/stevexia37/shibajian-bingqi/settings/pages
2. Source 选择 "Deploy from a branch"
3. Branch 选择 `main` / `/ (root)`
4. 点击 Save，等待 1-2 分钟部署完成
